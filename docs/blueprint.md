# **App Name**: Audio Blog AI

## Core Features:

- Audio Recording: Record audio directly within the app.
- AI Transcription: Automatically transcribe recorded audio to text.
- Blog Post Generation: Transform transcribed text into a structured blog post using AI; AI model will tool-use section headings, lists, etc. as appropriate.
- AI Style Training: Train an AI model using your past writing samples to mimic your unique style.
- Rich Text Editor: Edit the generated blog post using a rich text editor.
- Export Options: Export blog posts in various formats (plain text, HTML, Markdown).
- User Authentication: Allow users to create accounts and manage their blog posts.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a calm and creative atmosphere.
- Background color: Light beige (#F5F5DC) to provide a neutral, pastel backdrop.
- Accent color: Pale green (#98FF98) to highlight important actions.
- Body and headline font: 'PT Sans' (sans-serif) for readability and a modern, slightly warm aesthetic.
- Use minimalist, line-based icons that match the pastel color scheme.
- Keep the layout clean and simple with plenty of white space. Use a card-based design for organizing blog posts.
- Incorporate subtle animations, like fading effects and smooth transitions, to enhance user experience.
